## create git repo
```
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/somdevsheel/Algorithmic-Trading-Predictive-Models-for-Stock-Prices-using-Machine-Learning.git
git push -u origin main
```

## push an existing repository from the command line
```
git remote add origin https://github.com/somdevsheel/Algorithmic-Trading-Predictive-Models-for-Stock-Prices-using-Machine-Learning.git
git branch -M main
git push -u origin main
```

### git daily command 
```
git add . or git add <changed file name>
git commit -m "fill with any change"
git push
```
